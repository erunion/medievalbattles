Medieval Battles

A free browser based, MMORPG.

Download .zip Download .tar.gz View on GitHub

Medieval Battles was a browser based MMORPG that ran from 2002 until sometime in 2007. It was a lot of fun, but was plagued by my overall lack of experience with programming (I taught myself how to program through it) and managing a community.

It's now open source. The database is mostly there though you might run into some issues with a table (or two) or data missing.

I've thought about reviving it, but just haven't had the time.

If you spin up an instance, I'd love to hear about it.
